A simple Todo application built with React Native and TypeScript.

## Features
- Add / Update / Delete Todos
- Biometric authentication using Expo Local Authentication
- Clean UI with bottom input
- Unit tests for authentication logic

## Tech Stack
- React Native (CLI)
- TypeScript
- Expo Local Authentication
- Jest


# Getting Started

> **Note**: Make sure you have completed the [Set Up Your Environment](https://reactnative.dev/docs/set-up-your-environment) guide before proceeding.


## Project Setup

### Prerequisites

Make sure the following are installed:

- Node.js (LTS)
- npm
- Android Studio (SDK + Emulator)
- Java 17
- Android device or emulator

---

## ðŸš€ Getting Started

### 1: Install Dependencies  
Run from the **project root directory**:

```bash
npm install
```
### 2: Start Metro Bundler
Run in a separate terminal from the project root:

```bash
npx react-native start
```
### 3: Run App on Android
Make sure an emulator is running or a device is connected:

```bash
npx react-native run-android
```

This will build and install the app on the device.

### 4: Run Unit Tests

```bash
npm test
```
Runs Jest tests
